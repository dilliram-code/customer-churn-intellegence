# 📊 Customer Churn Prediction (End-to-End ML Project)

This project is a complete **end-to-end machine learning solution** to predict customer churn. It covers everything from data preprocessing to model deployment, helping businesses identify customers who are likely to leave.

---

## 🚀 Project Overview

Customer churn prediction is a crucial task for businesses aiming to improve customer retention. In this project, we:

- Clean and preprocess raw data
- Perform exploratory data analysis (EDA)
- Apply feature engineering
- Train multiple machine learning models
- Evaluate model performance
- Deploy the best model using **Streamlit**

---

## 🧠 Models Used

The following models are trained and compared:

- **Logistic Regression**
- **Random Forest Classifier**
- **XGBoost Classifier**

---

## ⚙️ Tech Stack

- Python 🐍
- Pandas & NumPy
- Scikit-learn
- XGBoost
- Matplotlib & Seaborn
- SHAP (for model explainability)
- Streamlit (for deployment)

---

---

## 📈 Workflow

1. Data Collection  
2. Data Cleaning & Preprocessing  
3. Exploratory Data Analysis (EDA)  
4. Feature Engineering  
5. Model Training  
6. Model Evaluation (Accuracy, Precision, Recall, F1-score, ROC-AUC)  
7. Model Explainability using SHAP  
8. Deployment with Streamlit  

---

## 📊 Model Evaluation

Models are evaluated using:

- Accuracy
- Precision
- Recall
- F1 Score
- ROC-AUC Score

The best-performing model is selected based on these metrics.
